# Any2All — File Conversion Hub

A production-style Next.js 14 app (App Router) with Tailwind CSS, SEO-ready metadata, ad slots, and three working client-side tools:
- PNG → JPG
- JPG → PDF
- Compress Image

## Quick start

```bash
npm install
npm run dev
```

Deploy on Vercel and add your custom domain (any2all.net).

## Ads
Replace the `AdSlot` component with your AdSense snippet once approved.
